# 🚗 Консольное приложение "Автосалон ТОП"

Учебный проект, разработанный в рамках 2-недельной производственной практики. Программа имитирует работу интерактивного автосалона с возможностью управления пользователями, а также покупки и продажи транспортных средств.

---

## 📋 Функционал программы

1. Регистрация пользователей с проверкой ввода данных.
2. Просмотр каталога доступных автомобилей и мотоциклов.
3. Покупка транспорта (с автоматической проверкой баланса покупателя).
4. Возврат транспорта в автосалон с выплатой 85% от его первоначальной стоимости.
